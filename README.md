# Employee & Customer Management System

A full-stack Employee & Customer Management System built with **Django REST Framework** and **React.js**. The application provides secure authentication, employee and department management, analytics dashboards, and a modern responsive user interface.

## 🚀 Features

* Secure JWT Authentication
* Employee Management (CRUD)
* Department Management (CRUD)
* Search, Filter, Sort & Pagination
* Employee Profile Image Upload
* Analytics Dashboard with Charts
* Export Employee Data to PDF & Excel
* Responsive User Interface
* Toast Notifications & Confirmation Dialogs
* RESTful API Architecture
* PostgreSQL Database Support
* Production Deployment (Render & Vercel)

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/employee-customer-management.git
```

```bash
cd employee-customer-management
```

### Backend Setup

```bash
cd backend
```

Create and activate a virtual environment.

Windows

```bash
python -m venv env
env\Scripts\activate
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Apply migrations.

```bash
python manage.py migrate
```

Run the backend server.

```bash
python manage.py runserver
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
